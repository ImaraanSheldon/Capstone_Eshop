API Endpoints Documentation

Users API Endpoints

Description Method Endpoint
Fetch all users GET /users
Fetch a single user by ID GET /users/:id
Register a new user POST /users/register
Update a user by ID PATCH /users/:id
Delete a user by ID DELETE /users/:id
Login a user POST /users/login

Games API Endpoints

Description Method Endpoint
Fetch all games GET /games
Fetch games published by Nintendo GET /games/Nintendo_Published
Fetch games developed by Nintendo GET /games/Nintendo_Developed
Search games GET /games/search
Fetch games with discounts GET /games/discounts
Fetch recent games GET /games/recent
Fetch best sellers (charts) GET /games/charts
Fetch a single game by ID GET /games/:id
Register a new game POST /games/register
Update a game by ID PATCH /games/:id
Delete a game by ID DELETE /games/:id
